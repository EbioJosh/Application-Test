Raspberry Pi Banking Terminal - Frontend Test

Frontend Structure

This directory contains the React frontend for the Raspberry Pi banking terminal.

  • App.js - Main application component
  • App.css - Styling for the application
  • index.js - Entry point
  • package.json - Dependencies and scripts

To Build and Run

  1. Navigate to this directory in your terminal
  2. Run npm install to install dependencies
  3. Run npm start to start the development server
  4. Run npm run build to create a production build

Integration with Backend

The frontend communicates with the Flask backend through:

  • REST API endpoints for data operations
  • WebSocket connections for real-time hardware events
